# 2026 岡山・神戶行 Dashboard

這個專案是手機優先的旅遊 Dashboard，可直接部署到 GitHub Pages。

## 結構

- `index.html`：頁面入口
- `styles.css`：版面與手機樣式
- `app.js`：渲染邏輯
- `data/trip-data.js`：公開旅遊資料
- `private/vjw-accounts.md`：僅本機保存的敏感資訊，不會被 Git 追蹤

## 目前已整理內容

- 航班資訊
- 待辦清單
- 交通 / JR Pass
- 網路方案
- 天氣查詢
